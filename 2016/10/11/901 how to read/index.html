阅读小结

发表于 2016-10-11   |  

个人总结，作用在于梳理个人思维系统，形成有意识有规律的行为模式，然后遇到新的知识就可以从容往里套。说到底，做好一件事的关键“但手熟尔”，不过手熟的方法不是漫无目的的尝试，而是一点技巧及长时间的刻意练习。

关于读书，主要需要回答以下两个问题：

如何选书

选书，从途径上来说可以分为人事网书四种，找你要了解问题的专业人士推荐或者同好推荐，就事论事的找工具书，利用网站论坛筛选优秀的书，或者在阅读某本书时又提到了一些作者或书可能会比较对胃口。

不过，我现在的情况不是找不到书看，而是想读的太多，读过的太少，所以在筛选下一本书的时候，主要有如下原则：

  1. 读书要有一定目的、方向和立足点，可以一段时间设定一个读书的主题，然后找相关的书形成主题阅读，最好能梳理出一个知识体系，方便后续的补完，另外，再单独找一本偏娱乐的书作为零碎时间的消遣；
  2. 要读通几部大部头作为最基本的东西，建立知识体系的基础和完整框架；
  3. 读一流学者的一流书，尝试读原典，尝试读读哲学书；
  4. 快速筛选原则，在建立基本的知识概念之后，再选书的时候，可以看看与书相关的书评、思维导图、目录，看看有没有自己感兴趣的话题，再决定是否看下去；
  5. 一本好书，是它真的能帮助你认识当下的世界与自己——刘瑜。

《越读者》中，郝明义则提到了“阅读与饮食” 这一概念，如果把饮食与阅读相对比，那么它们具有一些共性的东西：

  1. 主食阅读：主食是人们必不可少的食物，换做书籍来说就是生存所需的书，例如管理学、经济学、机械设计这些工作专业性很强的书籍。
    主食阅读在一定的生存条件下是必不可少的，阅读时会有较强的目的性。
  2. 美食阅读：美食是补充蛋白质的高营养食物，也就是给人舒适感的书，能带来思想深处的共鸣，比如哲学书籍或者文学名著等。
    在这类书籍中需要从某个角度去发现问题，探索书中涉及的现实问题，促使我们去思考与人生相关的问题。当然，有些书看起来美味可口，但不一定适合自己，所以阅读也要有选择性。
  3. 蔬菜水果类阅读：蔬菜水果补充纤维素的人体所需，在书籍上的定义就是工具书，比如字典、辞典类书。
  4. 甜食阅读：调节口感的食物，代表着娱乐休闲类的书籍。甜食阅读读起来爽，适合娱乐和消遣，追求快感。偶尔读没问题，但每天花四五个小时去看玄幻小说或者言情小说，是不营养的。

之所以将读书与饮食做比较，我想这也是由于作者希望读书的均衡性，每一类型的书都应该有所涉猎。这同样是我定阅读目标时的考虑，生存阶段，主食为主，再加少许美食和甜食打打牙祭。

如何读书

不同类型的书有不同的读法，这里我先说主食，也就是功利性较强的书的读法，其他类型的书也许可以借鉴，但往往我并不完全遵守。另外就是下面我罗列了一些方法，自己也还在尝试，一般来说，用一个方法阅读5-10本书就可以看出这个方法是否适合自己，后续有了新的概念，我也会持续添加进来。
通常， 阅读一本书可以依次有准备、阅读、思考、实践、分享、复习这六个步骤。思考着重理解，实践强调应用。而读实用的书，第一层境界是看懂别人的意思，第二层是自己能照着讲出来，第三层是自己有意识地去应用，最后一层是可以用自己的话总结出来变成自己的东西。

一. 准备

  1. 前面的筛选阶段利用网络查找的思维导图、书评等已足以建立一个大概的印象；
  2. 复习自己之前根据相关内容画的思维导图；
  3. 不同的书对于相同的概念所使用的词汇很可能会有一定的差异性，需要根据阅读主题建立一套比较中立的词汇，这套词汇会随着相关主题的阅读而逐渐清晰；
  4. 费曼技巧，其做法如下：拿出一张纸，在上面写下认为已经掌握的当前主题下的知识点名称，在下方写下对这个概念的理解，设想你要如何向别人解释清楚这个知识点，在写的过程中，通常会发现自己对一些知识点完全掌握不清，这个时间再去回顾书本，重新学习，一般来说在主题阅读中，重点的内容会不断重复出现
  5. 如果可能，思考一下哪些是重点需要解决的问题，哪些是次要的，分清轻重缓急。

二. 阅读

  1. 好的书至少读两遍，一遍通读，一遍针对有兴趣的地方精读并做读书笔记；
  2. 阅读时的注意力问题，阅读不同类型的书籍时注意力情况也不同，了解自己的注意力曲线和范围，不一定要长时间阅读，定期休息几分钟，安排好休息时间的内容。读累了就做个笔记、思维导图或挑着文字发条微博，或者活动一下。
    • 专业性强一点的书（教材、专业材料）,强迫自己能集中注意力20-30分钟（具体时间要看书的类型和当然的体力、心情）然后做笔记或思维导图，在重新清晰一下思路，有时要反复读上很多遍
    • 一些社科畅销书书，不会过于难懂，基本几天能阅读完第一遍，写简评，做笔记或思维导图，在觉得值得深挖的地方再多看几遍
  3. 阅读笔记的做法，可以有书中画线、做批注、APP软件、思维导图、音频笔记、RIA拆书法等。书中画线、做批注、APP软件是阅读中的过程，思维导图、音频笔记、拆书法折偏向于一本书阅读完之后。
    • 由于现在大部分书都是在Kindle上阅读的，标记可以很方便的完成，之后可以利用clippings导出整理
    • 思维导图是现在很流行的图像化整理方法，就类似我开头展示的图片，主要好处是帮助整理书本的论述框架，重点是用关键词和自己的话将书的意思表达出来
    • 音频笔记来自与战隼（我自己也还在尝试）。方法是在微信上搜索一下自己的ID，然后把自己的ID置顶，把本周的内容用录音的方式说出来。一分钟正常的语速会控制在160-200字之间，五分钟能说800-1000，练习在3-5分钟内把本周阅读的内容总结出来。 微信的录音一次只能60秒，需要进行3-5次，多听几次自己录下来的话，你会发现自己的总结和表达都有很大的欠缺。准备时要注意抓住重点，尽量把逻辑和结构体现出来，要考虑哪部分会吸引别人的注意力，多尝试几次。可以把这种方法进行练习，当你熟练掌握之后，可以很容易在几分钟内向别人介绍一本书或一个观点。个人感觉也是个练习口才的方法
    • RIA拆书法来自于赵周，没怎么注意，RIA指的就是Reading-Interpretation-Appropriation，看下来拆书法不关注全书的框架，而注重对细节的不断挖掘，是偏应用的学习方法。

三. 思考

分析整理，建立自己的框架，或者整合到之前相关主题的框架中，主动思考书的观点，同时也要了解对立的观点。
当阅读完一本书，应该能回答以下问题：

  1. 整体来说，这本书到底在说些什么？在谈这门知识的哪些具体问题？
  2. 哪些问题得到了解决？哪些问题还没有？
  3. 作者细部说了什么，怎么说的？（逻辑脉络）
  4. 这本书说的有道理吗？是全有道理还是部分有道理？作者在书中讲的内容是科学还是观点？如果是科学，是否得到了验证？如果是观点，是否值得相信？自己需要有自己的判断。
  5. 这本书和你有什么关系？你接下来要做的是什么？（心得体会）

对于书中比较重要的话，思考应该贯穿这句话的含义、原因、用法和它的来龙去脉，这就要求你对当前所读的内容有深刻的了解。

四. 实践

  1. 对于方法论的书，都可以实践并作记录，不足的地方进行改善（（具体来说可列出所有细化成行动的建议，设立开始日期并开始进行尝试，一般尝试10次左右就可以知道这种方式是否适合自己以及自己的环境，尝试的过程中要记录自己遇到的问题并进行总结思考：成功了吗？成功了可以继续深入，没成功要思考为什么没成功？如何改进？尝试次数多了，新知才能内化为自己所用。所以一定要实践！）；
  2. 有些类似是管理或者和别人沟通、演讲的，不一定马上有机会来实现，这时运用“思想体操”，你可以在大脑假想，遇到一个什么场景，可能别人会问什么问题，按书上的方法应该怎么去对待。你读完每一个知识块，你必须设想去实践他们，你的脑中不断地模拟实践的场景，以及实践会遇到的困难和收获；
  3. 还有情况是发现自己实在用不上，就需要反思自己的阅读计划;
  4. 学会一鱼多吃，同一本书，还可以从其他角度应用，比如三体，如果让你去写三体，你会怎么写？这难道不就是一个写作者的角度吗？果你把它拍成电影你会怎么拍？这不又多了一个拍电影的角度吗？如果要你把书里面的内容是用你的语言复述出来，你会怎么以老师的维度说？水滴的结构，从工程师角度想一想?

五. 分享

  1. 分享的方式有很多，用自己的语言来翻译、浓缩作者的观点都是输出的方式，确实好的书，综合摘录、思维导图、回答问题、心得体会写一篇书评；
  2. 浓缩读的书中特别有感触的观点，在微信或微博上发个朋友圈，或与朋友简单说一说，看你能不能在几分钟内把至少一点说明白（如果能有机会，以正式一些的方式与很多人分享，效果会更好）；
  3. 模拟演讲，想象其他人会对你有什么问题，你该如何作答。

六. 复习

  1. 进一步查询收集相关资料；
  2. 用不同的方式来学习阅读；
  3. 至少需要隔一段时间复习一次，在完成阅读笔记后顺手设置复习的提醒工作，一个月后复习笔记一次；
  4. 做完一次主题阅读后，有了自己的输出点，如果能分享一次，过了三到六个月再重新回顾内容或结构，复习效果会比较好。可以再重新读一读在主题阅读中比较重要的书籍

其他

P1 阅读记录：

前期对自己的整体阅读情况做下系统的了解，有助于养成更加自觉的阅读习惯，当然后面有了了解之后可以不再记录：

  1. 本周一共阅读了多长时间？ 对你来说，每天的阅读时间是长了还是短了，有必要调整吗？
  2. 阅读的时间段都分布在哪几个时间段? 觉得这些时间段里，哪个时间段阅读效果最好，哪个最差？下周需要改进吗？
  3. 尝试了几个阅读环境，在哪个阅读环境中效果最好？
  4. 注意过你在阅读时自己注意力能集中多长时间吗？一般会受到哪些干扰，下回应该如何避免？
  5. 在每天坚持阅读都遇到了哪些问题？ 想想怎么能让你坚持下去？
  6. 对自己一周的阅读内容做个总结？都读了哪些内容，有什么收获？尝试一下用5分钟或10分钟对本周的心得做下总结，想想哪些内容可以应用或触动。用录音试，自己听听，然后找机会讲给别人听。写篇文章或短文，总结一下，分享出来。

P2 新知识与已有知识建立联系的方法

  1. 搜寻类似知识，建立关联
  2. 用最熟悉的知识建立关联
  3. 用具有相似结构的知识体系建立关联
  4. 一次建立关联后，多次巩固，关联的绳索才能粗壮
  5. 拆解分析旧知识的结构，拆成粉末再组合，加深熟悉程度
  6. 把知识当做自家人，多次使用它，它便会成为你最牢固的知识
  7. 绞尽脑汁无法建立关联，那是你的旧知识太少，或者你根本不了解你的旧知识，在需要的时候使不上力
  8. 别羡慕别人的知识，他们都是一步一步学出来的，你硬要一步登天学会他们会的，就等着一嘴毛
  9. 思考并不是由你来完成的，认清这个现实吧，你只是告诉大脑要搜索类似的知识，他们便会去搜索，而不是你来搜索你的知识库存。简单的说，你只是发布了个命令而已，具体的思考活动是你的潜意识做的
  10. 学习新知识，而不是简单的收藏
  11. 时常复习回顾已有的知识
    看下来就是知识管理相关的内容了，这个以后有时间再学习。
